	O que é:
Engenharia de Software é a aplicação de princípios de engenharia e ciência da computação para projetar, desenvolver, testar, manter e gerenciar sistemas de software de alta qualidade, focando em confiabilidade, eficiência e escalabilidade em todo o seu ciclo de vida, desde a concepção até a implementação e evolução, abrangendo desde apps móveis até sistemas corporativos complexos, envolvendo metodologias ágeis, linguagens de programação e gestão de projetos para criar soluções robustas para o dia a dia e negócios.

	Ligação com ads
É uma matéria que caminha em igual com ads, seu começo se entrelaça em vários aspectos, principalmente utilizando linguagens de programação e etc.

	Banco de dados
É a ligação direta com o sistema, se tornando a matéria fundamental para o analista.
