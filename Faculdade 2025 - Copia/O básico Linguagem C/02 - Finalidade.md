A finalidade da linguagem C é fornecer uma linguagem de programação de baixo nível, eficiente e portável para criar softwares de sistema, como sistemas operacionais (Linux, Windows), drivers, e aplicações embarcadas, permitindo controle direto do hardware, além de ser uma base fundamental para o aprendizado de outras linguagens e para desenvolver sistemas críticos onde performance e controle de memória são cruciais
